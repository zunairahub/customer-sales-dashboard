# Customer Sales Dashboard — Superstore Dataset

Interactive sales analytics dashboard built in Python identifying revenue trends and top performers.

## Key Insights
- Identified top performing region, category, and customer segment
- Monthly revenue trend analysis across full date range
- Top 10 products and customers by total sales
- Sub-category breakdown for targeted business decisions

## Tools Used
Python · Pandas · Matplotlib · Seaborn · Kaggle
